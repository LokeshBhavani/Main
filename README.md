### 🚀 Pinned projects worth a look

**SynthMind** — ESP32-S3 AI desktop companion robot
- Dual-MCU architecture (ESP32-S3 for brains, ESP32-C3 for motor/servo control via TB6612FNG)
- I2S audio in/out (INMP441 mic + MAX98357A amp), servo-driven facial expressions, RGB feedback
